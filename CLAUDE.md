# Guild of Adventurer Loyalty Platform

## Design Context

### Users
- **Admins**: Club managers of the "Guild of Adventurer". They need a simple, efficient way to manage members, loyalty configurations, and club activities.
- **Adventurers**: Members of a role-playing game club (D&D, Cthulhu, etc.). They interact with the system via a Telegram Mini-App to track their "Adventurer Level" and progress.

### Brand Personality
- **Voice & Tone**: Friendly, Modern, and Experiential.
- **3-Word Personality**: Friendly, Modern, Adventurous.
- **Emotional Goals**: To make the administrative side of a hobby feel like part of the adventure, while remaining functional and professional.

### Aesthetic Direction
- **Visual Tone**: Modern design with subtle, high-quality RPG/Fantasy influences (D&D, Witcher). Not an overt "fantasy" skin, but a sophisticated integration of RPG motifs and modern UI patterns.
- **Color Palette**: OKLCH-based premium dark mode.
    - **Primary Neutral**: Deep Black (`#161616`)
    - **Accent/Brand**: Gold/Parchment gradients (`#f4e5c4` to `#c8a679`).
- **Typography**: Modern, readable fonts (specified in CSS variables).
- **Themes**: Primarily focused on a premium dark mode, reflecting a "tavern" or "guild hall" vibe.

### Design Principles
1. **Adventurous Utility**: Every administrative task should feel efficient but never sterile. Use subtle animations and micro-interactions to add "adventurer" flair without slowing down the user.
2. **Modern Legacy**: Blend modern UI patterns (Shadcn, Radix) with classic RPG aesthetics (gold accents, parchment tones).
3. **Progressive Gamification**: Surface the concept of "Adventurer Levels" and progression throughout the UI where appropriate to reinforce the club's purpose.
4. **Ukrainian-First**: The primary interface language is Ukrainian. All new components and copy must respect the cultural context and localization standards set by the existing codebase.
